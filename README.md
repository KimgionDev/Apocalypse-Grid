# Apocalypse Grid
Mô tả: Đồ án Niên luận Cơ sở ngành KTPM. Trò chơi 2D Top-down Shooter phát triển bằng Unity. Trọng tâm: Ứng dụng thuật toán phân chia không gian nhị phân (BSP Tree) để tạo màn chơi ngẫu nhiên (Procedural Dungeon Generation).

Description: 2D Top-down Shooter developed in Unity for the Software Engineering Foundation course. Core focus: Implementing Binary Space Partitioning (BSP Tree) for Procedural Dungeon Generation.
